# Transcription Frontend

This is the frontend application for the Multimodal Transcription & Local LLM system.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm run build`

Builds the app for production to the `build` folder.

### `npm test`

Launches the test runner in interactive watch mode.

## Features

- Upload audio/video files for transcription
- Process YouTube links
- Extract text from web pages
- Chat with local LLM
- Download transcription results

## Configuration

Make sure the backend API is running on `http://localhost:8000` or update the `API_URL` in `src/App.jsx`.
