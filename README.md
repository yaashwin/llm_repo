# Multimodal Transcription & Local LLM Application

A comprehensive application for transcribing audio, video, YouTube links, and web pages, with integrated local LLM chat functionality.

## Features

- **Multiple Input Types**: Support for audio files, video files, YouTube links, and web URLs
- **High-Quality Transcription**: Using OpenAI's Whisper model for accurate transcription
- **Local LLM Integration**: Chat with local language models using Ollama
- **User-Friendly Interface**: Modern, responsive web interface
- **Download Options**: Export transcriptions as text files
- **Real-time Processing**: See results as they're generated

## Prerequisites

- Python 3.8+
- Node.js 14+
- FFmpeg (for audio/video processing)
- Ollama (for local LLM)
